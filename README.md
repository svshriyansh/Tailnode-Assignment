## Prerequisites

> .env
> Create a .env file inside both folders i.e. Part A and Part B and save your MongoDB credentials in it as
>>
```
DB_PASSWORD=your_password
DB_USER=your_username
```
  
 <h3 align ="center">
   Part A
 </h3>

## Run Command
> 
```
  pip3 install -r requirements.txt 
```

## To run script
> 
 ```
   python3 script.py 
```


 <h3 align ="center">

 </h3>
 
## Run command
